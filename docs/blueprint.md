# **App Name**: OSHIQLAR OYINI

## Core Features:

- Secure Auth System: User login and registration screens to maintain profile and balance history.
- Betting Hub: Dashboard displaying User ID, balance in so'm, and preset betting tiers (40k to 1.2m so'm).
- Automated Matchmaking: Simulation tool that finds opponents and prompts users with a 20s acceptance timer.
- AI Dice Simulation: An AI-powered tool to simulate the 'Tosh tashlash' dice roll logic against an intelligent bot.
- Balance Ledger & Tax Logic: Persistent record-keeping of user balances including a 2% automated company tax on winnings.
- Admin Management Console: High-level dashboard for administrators to monitor user stats and distribute manual balance top-ups.

## Style Guidelines:

- Primary color: Lustrous Gold (#D4AF37) used for buttons and primary headers to evoke luxury and high stakes.
- Background: Midnight Bronze (#161512), a heavily desaturated deep dark tone that provides a premium canvas.
- Accent: Burnt Copper (#A34D29) for highlighting active states and specialized UI alerts, providing analogous contrast to the gold.
- Headline font: 'Playfair', an elegant high-contrast serif for titles. Body font: 'PT Sans' for readable, modern interface elements.
- Stylized, minimalist icons for traditional elements like 'oshik' pieces, rendered in metallic gold outlines.
- Centrally focused, card-based mobile responsive layout prioritizing betting buttons for easy thumb access.
- Physics-based 3D rotation animations for dice rolling and smooth progressive count-ups for balance changes.
